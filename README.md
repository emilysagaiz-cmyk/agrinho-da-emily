
- **index.html**: Estrutura HTML do site  
- **style.css**: Estilos e layout, incluindo responsividade e animações  
- **script.js**: Funções interativas, animações ao rolar a página e alerta do botão "Saiba Mais"  
- **img/**: Pasta com todas as imagens usadas no site  

---

## Funcionalidades

- **Hero Section** com logo, título, descrição e botão de orçamento  
- **Seção do Comedouro** com imagem e lista de benefícios  
- **Cards de Benefícios** com efeito hover  
- **Galeria de Imagens** com destaque para banner, comedouro e maquete do projeto  
- **Formulário de Contato** com validação e alerta de envio  
- **Footer** com informações do projeto  
- **Animações suaves** ao rolar a página  
- **Layout responsivo** para celular, tablet e desktop  

---

## Tecnologias

- HTML5  
- CSS3  
- JavaScript  
- Fonte: [Poppins](https://fonts.google.com/specimen/Poppins)  

---

## Como Executar

1. Baixe ou clone o projeto.  
2. Mantenha todos os arquivos na mesma pasta, incluindo a pasta `img`.  
3. Abra o arquivo `index.html` em qualquer navegador moderno.
