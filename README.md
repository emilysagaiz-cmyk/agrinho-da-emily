# Agrotec Sustentável - Comedouro Inteligente

Este é um projeto de Landing Page focado em tecnologia aplicada ao campo de forma sustentável, apresentando soluções de automação para o manejo animal com foco na otimização de recursos e redução do desperdício de insumos.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Design responsivo, variáveis nativas e animações fluidas.
* **JavaScript (ES6):** Manipulação dinâmica do DOM para o menu mobile e efeitos de surgimento em tela baseados na API `IntersectionObserver`.
